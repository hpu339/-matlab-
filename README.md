# 基于matlab的数字信号降噪系统以及GUI界面

### 综合文件介绍,所有文件放到一块避免无法调用
#### 1.untitled.m和untitled.flg相关=关联。其中需要调用到其他函数比如ideal_lp.m,filter_hpu.m
#### 2.AWGN.m为高斯白噪声噪音文件
#### 3.noise.m为特定频率噪声的文件
#### 4.add_noise.m为加入噪声以后的文件
#### 5.FIR_DF.m为FIR滤波器文件
#### 6.IIR_DF.m为IIR滤波器文件
#### 7.两个.wav文件为两个音频文件
